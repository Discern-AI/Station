
# Community Beta

Community Beta adds the first real public social layer to Station:

- Discover feed scaffolding
- search route scaffolding
- forum categories
- thread creation/read scaffolding
- comments on documents and threads
- moderation report route scaffolding

This build is still mock-data-first. The next production step is wiring these routes and pages to Supabase-backed persistence, tier checks, and moderation tools.
