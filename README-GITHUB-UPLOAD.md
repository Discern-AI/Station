# Station GitHub Upload Bundle

This bundle is the latest available Station scaffold from the chat session, packaged so you can upload it directly into a new GitHub repository.

## What it is
- monorepo scaffold
- Next.js web app shell
- Express API shell
- shared packages
- Studio, public Spaces, Discover, forums, comments, and moderation scaffolding

## What it is not yet
- production-ready
- wired to real Supabase persistence
- wired to real Stripe billing
- wired to live DeepSeek requests

## Quick upload steps
1. Create a new empty GitHub repository.
2. Extract this zip locally.
3. Copy the extracted files into your repo folder.
4. Commit and push.
5. Open the repo in your editor and follow README.md plus the Supabase / Stripe / DeepSeek plans from the chat.

## Important
This is a scaffold intended to be wired up next, not a finished deployed application.
